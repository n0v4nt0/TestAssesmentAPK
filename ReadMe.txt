Project Dibuat Menggunakan:

- Visual Studio 2019
- .Net Core 3.0
- Sql Server 2019

Dengan Package Dan Namspace:

- EntityFramework 6.4.4
- EntityFrameworkCore.SqlServer 3.1.8
- EntityFrameworkCore.Tools 3.1.8
- CodeGeneration.Design 3.1.4
- Azure.Container 1.11.1

Tata Cara Penggunaan:
- Siapkan VS 2019 dan Sql Server 2019 atau keatas
- Klik dua kali pada file dengan kata TestToko.sln
- Buka folder Sql Server dan jalankan setiap file kecualai file bertuliskan TokoDB.bak karena itu backup databasenya
- setelah sudah berhasil semua maka run program dengan vs 2019
